The Properties page is the main operating list for the portfolio.

## What the Directory Shows

- Property name, type, and address.
- Unit count and occupancy percentage.
- Monthly rent and overdue balance.
- Leases expiring in the next 90 days.
- Open maintenance ticket count.
- Property status and row actions.

## Search Properties

Use the search field to find properties by name, city, state, property type, or status.

## Filter Properties

Select **Filters** to narrow the directory by property, city, state, zip code, status, minimum unit count, open maintenance requests, or archived status.

Select **Apply Filters** to update the list or **Clear Filters** to reset it. When filters are active, the Filters button shows the active filter count.

## Expand Units

For multi-unit properties, select the expand control next to the property name. Expanded unit rows show unit-level occupancy, rent, overdue balance, lease expiration status, maintenance ticket count, and current status.
