title: Manage a Property
---

Each property row includes an overflow actions menu.

## Edit a Property

Use **Edit** to update property details, address, unit count, type, or contact information.

The property drawer also includes **Unit Details**. Use it to add, edit, or remove unit cards associated with the property. Existing units appear in the edit drawer when they are available.

## Remove a Unit

Use the trash action on a unit card to remove a unit from the property. Parcelis asks for confirmation before removing the unit from the edit form.

## Add Notes

Use **Add Notes** to save internal property notes. Notes are intended for operational context such as ownership preferences, access details, renewal reminders, or property-specific instructions.

## Archive a Property

Use **Archive** to hide a property from the default property list without permanently deleting it. Archived properties can be found by filtering for archived records. The archive action is unavailable for properties that are already archived.

## Delete a Property

Use **Delete** only when the property should be permanently removed. Parcelis asks for confirmation before deleting.
