---

Use the **Unit Details** step in the property drawer to review, add, edit, or remove units for a property.

## Unit Cards

Each unit appears as a card with these primary fields:

- Unit name.
- Market rate.
- Unit type: Residential or Commercial.

The card header also shows bedroom, bathroom, and square-foot summary chips. Empty values display as `0`.

## Expanded Details

Select **Unit Details** in a unit card to expand additional fields:

- Bedrooms.
- Bathrooms.
- Square feet.
- Rent includes.
- Amenities.

Rent includes and amenities are loaded from Parcelis unit option records.

## Add a Unit

Use **Add Unit** to add another unit card to the property. New units must have a unit name, market rate, and unit type before the property can be saved.

## Remove a Unit

Use the trash action on a unit card to remove it from the property. Parcelis asks for confirmation before removing the unit from the edit form.
