title: Keyboard Shortcuts
---

Use `Cmd` on macOS or `Ctrl` on Windows and Linux for shortcuts labeled `Cmd/Ctrl`.

| Shortcut | Action | Available when |
| --- | --- | --- |
| `Cmd/Ctrl+Shift+P` | Open or close the property drawer. | On the Portfolio dashboard and Properties page. |
| `/` | Focus property search. | On the Properties page while the property drawer is closed. |
| `Cmd/Ctrl+Enter` | Submit the property drawer. | The drawer is open, all required fields are valid, and the form is not submitting. |
| `Cmd/Ctrl+Shift+D` | Toggle between light and dark theme modes. | Anywhere in Parcelis. |
