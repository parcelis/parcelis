Parcelis helps property teams manage a portfolio from a single workspace. Use this guide to understand the platform, set up your first property, and work through day-to-day property operations.

Start with **Getting Started**, then use **Core Concepts** to understand the records in Parcelis. **App Functionality** contains the detailed workflow guides for adding, finding, managing, and reviewing properties.

## Current Product Scope

The current platform supports a portfolio dashboard, property creation and editing, property search and filters, property notes, archive and delete actions, and unit, lease, maintenance, and financial summaries when data exists.

The sidebar also shows planned areas for Leases, Tenants, Maintenance, Accounting, and Settings. These are not yet full standalone workflows.
